# App-TimmoS

Application immobilière professionnelle développée pour analyser le marché, estimer des biens et exploiter les données publiques.

## 🚀 Fonctionnalités

- 📊 DVF : consultation des biens vendus
- 🌱 DPE : affichage des performances énergétiques
- 🗺️ Cadastre : visualisation des parcelles
- 🏘️ Annonces : analyse du marché actuel
- 🧠 Estimation : calcul de valeur d’un bien immobilier

## 🛠️ Stack technique

- Next.js (App Router)
- TypeScript
- Leaflet (cartographie)
- Axios
- Vercel (déploiement)
